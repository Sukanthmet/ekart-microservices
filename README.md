# eKart Microservices Project

This project is developed using Spring Boot and Microservices architecture.

## Services

- Product Service
- Order Service
- Inventory Service
- Notification Service

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Apache Kafka
- REST API
- Maven

## Features

- Product CRUD operations
- Order Management
- Inventory Management
- Kafka-based event communication
- Global Exception Handling
- Validation
